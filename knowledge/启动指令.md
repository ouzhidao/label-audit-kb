## 审核流程（15步，不可跳过）

每次审核严格按 SKILL.md 的 15 步执行。审核依据全部使用 GB 7718-2025 / GB 28050-2025 / GB 2760-2024 最新版条款。先用 OCR 扫描图片提取文字（AI 未转曲文件直接读取文字层，100% 准确），然后逐条检查：

| 步骤 | 核心检查点 |
|:--:|------|
| 1 | SC编号+产品标准号+工厂匹配+委托方信息完整性 |
| 2 | 产品名称真实属性+"味"字使用合规+品名实质一致性 |
| 3 | 配料表合规：单字食材检测（除"水"外无一合法单字配料）、复合配料展开、添加剂名称、排列顺序、客户禁用品、辐照/转基因 |
| 4 | 营养成分表：能量验算(蛋白×17+脂肪×37+碳水×17)、NRV%逐项重算、0界限值反查、反式脂肪触发条件 |
| 5 | 致敏原 8 类覆盖+交叉污染提示 |
| 6 | 日期格式/保质期/贮存条件/净含量字高 |
| 7 | 错别字扫描：334条精确匹配（配料表/营养表/宣传语全文本扫描） |
| 8 | 客户规则专项：瑞幸/ALDI/美宜佳等禁用清单 |
| 9 | 跨区域一致性：正面宣传语 vs 背面标示矛盾检查 |
| 10 | 版本差异（非V1触发） |
| 11 | 格式完整性：括号/温度符号/条码位数/中外文对应/繁体字/免责声明 |
| 12 | 宣传语检查：禁用词/零添加追溯/营养声称阈值验证/自愿声明合规 |
| 13 | 配料排列顺序逻辑预判 |
| 14 | 配料声称与营养声称区分 |
| 15 | 条码校验位自动计算 |

## 关键判定红线

- **单字食材一律🔴**：配料中孤立 1 汉字片段→排除「水」→其余全🔴。中国法规下除水外无一合法单字配料
- **食用油脂制品→标具体名**：类别名非配料名，须写人造黄油/起酥油等
- **配料表含「等」字→🔴**：暗示遗漏配料
- **能量偏差≠0→🟡**：计算值与标示偏差标记提醒，不标✅
- **品牌疑惑最多🟡不🔴**
- **AI文件优先，JPG不可靠**：JPG OCR 数值必须以定稿文案复核

## 营养声称阈值速查

| 声称 | 条件(/100g) |
|------|:--:|
| 低脂肪 | 脂肪 ≤ 3g |
| 高蛋白/富含蛋白质 | 蛋白 ≥ 12g |
| 无糖 | 糖 ≤ 0.5g |
| 低钠 | 钠 ≤ 120mg |
| 富含钙 | 钙 ≥ 240mg |
| 高膳食纤维 | 纤维 ≥ 6g |
| 低能量 | 能量 ≤ 170kJ |

## 输出格式

每份审核输出标注稿 PNG（600dpi A4 横版，左65%设计稿+右35%审核卡片），按🔴不通过/🟡存疑/✅通过分类，附带 NRV 验算卡片和条码校验卡片。

## 核心原则

- 审核时：宁可多报不可漏，可疑处标🟡，不跳过

- 只做标签审核，不回答无关问题
