# 广告法 — 食品标签审核关联条款

> 现行版本：《中华人民共和国广告法》2021年修正版
> 配合：GB 7718-2025 第3.4/3.5条、第7章食品声称

---

## 第9条 — 广告禁止情形（标签审核核心）

下列情形不得在标签中出现：

| 禁止内容 | 示例 | 判定 |
|------|------|:--:|
| 使用"国家级""最高级""最佳"等绝对化用语 | "全国销量第一""最佳口感""首选品牌" | 🔴 |
| 使用国旗/国徽/军旗等 | 标签图案含国旗元素 | 🔴 |
| 使用国家机关/工作人员名义 | "国务院推荐" | 🔴 |

### 绝对化用语清单（食品标签高频风险）

| 禁用词 | 同义词/变体 |
|------|------|
| 第一 | 首选、No.1、冠军 |
| 最 | 最佳、最好、最优、最美 |
| 级 | 顶级、国家级、世界级 |
| 唯一 | 独家、仅有 |
| 极 | 极品、极致、绝佳 |
| 全网 | 全国、全球 |
| 100% | 百分之百、纯天然 |
| 零 | 零添加、0添加、零负担 |

> ⚠ "纯天然""零添加""不添加"同时受GB 7718-2025 4.4.2.2约束

## 第17条 — 食品广告特别规定

| 禁止行为 | 示例 |
|------|------|
| 含有治疗/疾病预防内容 | "养胃""护肝""改善便秘" |
| 声称具有保健功能（非保健食品） | "增强免疫力""延缓衰老""减肥" |
| 与药品/医疗器械混淆 | "堪比药物""一吃就见效" |

---

## 第28条 — 虚假广告

| 类型 | 标签场景 |
|------|------|
| 商品不存在/与描述不符 | 挂牛头卖马肉，标签配料造假 |
| 性能/功能/产地等虚假 | 国产标进口、含转基因标非转基因 |
| 使用虚构/伪造证据 | "经XX机构认证"但无证书 |
| 虚构使用效果 | "XX万人见证" |

---

## 互联网广告管理办法 (2023)

| 适用 | 要点 |
|------|------|
| 数字标签 | 二维码/小程序所链接的页面内容视为广告延伸 |
| 直播带货 | 标签上扫二维码进直播间→直播内容也受广告法约束 |

---

## 审核联动

| 标签要素 | 对应法规 |
|------|------|
| 宣传语/广告语 | 广告法第9/17/28条 + GB 7718-2025 3.4/3.5 |
| "不添加"用语 | 广告法第28条 + GB 7718-2025 4.4.2.2 |
| 保健功能声称 | 广告法第17条 + 食品安全法第73条 |
| 绝对化用语 | 广告法第9条 + 错别字库第七类 |
| 数字标签内容 | 广告法 + 互联网广告管理办法 |
