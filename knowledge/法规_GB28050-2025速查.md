# GB 28050-2025 营养标签审核速查

> 来源：GB 28050-2025 食品安全国家标准 预包装食品营养标签通则
> 发布：2025-03-16 | 实施：2027-03-16
> 替代：GB 28050-2011

---

## 2025版主要变化

- 增加了强制标示的营养素和提示语
- 修改了能量和部分营养成分的术语和定义
- 修改了部分营养成分含量的允许误差范围
- 修改了豁免强制标示营养标签的范围
- 增加了第8章（营养标签的呈现）
- 修改了部分营养素参考值(NRV)
- 修改了营养标签格式
- 修改了部分营养声称标准用语
- 增加了附录E"预包装食品份量参考值的推荐"

---

## 营养成分表强制标示内容

### 核心5项（必须标示）
| 顺序 | 营养素 | 单位 | 修约间隔 |
|:--:|--------|------|:--:|
| 1 | 能量 | 千焦(kJ) | 1 |
| 2 | 蛋白质 | 克(g) | 0.1 |
| 3 | 脂肪 | 克(g) | 0.1 |
| 4 | 碳水化合物 | 克(g) | 0.1 |
| 5 | 钠 | 毫克(mg) | 1 |

### 2025版新增强制标示
- **糖**：在碳水化合物下方标示
- 饱和脂肪酸（在脂肪下方）
- **提示语**：引导消费者合理膳食的提示

| 营养素 | "0"界限值(每100g) |
|--------|:--:|
| 能量 | ≤17kJ |
| 蛋白质 | ≤0.5g |

| 饱和脂肪 | ≤0.1g |
| 反式脂肪 | ≤0.3g |
| 碳水化合物 | ≤0.5g |
| **糖** | **≤0.5g** |
| 钠 | ≤5mg |

## 营养声称速查

### 含量声称阈值（最常用）

| 声称 | 条件（固体/100g） |
|------|------|
| 低脂肪 | ≤3g |
| 无脂肪 | ≤0.5g |
| 低糖 | ≤5g |
| 无糖 | ≤0.5g |
| 高蛋白 | ≥12g（≥20%NRV） |
| 高钙 | ≥240mg（≥30%NRV） |
| 低钠 | ≤120mg |
| 无钠 | ≤5mg |
| 高膳食纤维 | ≥6g |
| 低能量 | ≤170kJ |

## 审核要点

1. 能量计算验证：蛋白质×17 + 脂肪×37 + 碳水化合物×17 （如有膳食纤维，＋膳食纤维×8）
2. NRV% 逐一验算
3. "0"界限值判定
4. 营养声称与数值一致性（如宣称"高钙"→钙≥240mg/100g）
5. 格式：方框表、表头顺序
6. 单位：能量kJ、钠mg、其余g
