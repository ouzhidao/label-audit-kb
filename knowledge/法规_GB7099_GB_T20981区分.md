# GB 7099-2015 vs GB/T 20981-2021 区分指南

> 来源：标准原文 + 公司质量部门确认
> 关键认知：两者不是替代关系，是安全底线+品质上限的互补关系

---

## GB 7099-2015 食品安全国家标准 糕点、面包

| 属性 | 内容 |
|------|------|
| 标准类型 | **强制性国家标准（GB）** |
| 适用品类 | 糕点、面包全品类 |
| 管辖内容 | **安全底线**：微生物限量、重金属污染物、食品添加剂、酸价/过氧化值 |
| 标签标示 | 必须标示，不标违法 |
| 烘焙产品通用 | ✅ 调理面包、起酥面包、软式面包、烘烤糕点等全部适用 |

## GB/T 20981-2021 面包质量通则

| 属性 | 内容 |
|------|------|
| 标准类型 | **推荐性国家标准（GB/T）** |
| 适用品类 | 仅限面包（不含糕点） |
| 管辖内容 | **品质等级**：感官要求、水分、酸度、比容、口感 |
| 标签标示 | 自愿标示，不标不违法 |
| 适用子类 | 软式面包、硬式面包、起酥面包、调理面包等 |

---

## 审核时的判定规则

| 场景 | 判定 |
|------|:--:|
| 标签标 GB 7099 + 面包类产品 | ✅ 合规 |
| 标签标 GB/T 20981 + 面包类产品 | ✅ 合规（自愿） |
| 标签标 GB 7099 + 糕点类产品 | ✅ 合规 |
| 标签标 GB/T 20981 + 糕点类产品 | ❌ 不适用（20981 仅限面包，不含糕点） |
| 标签同时标 GB 7099 + GB/T 20981 | ✅ 合规（安全+品质双标） |
| 标签只有 GB/T 20981 没有 GB 7099 | ❌ 缺少强制安全标准 |
| 标签标 GB 7099 + 产品类别起酥面包 | ✅ **合规**（之前我理解错了） |
