# 供应商配料交叉验证规则

> 适用场景：用户提供了成品标签 + 供应商原始标签 + 各配料添加量
> 审核逻辑：8维度交叉验证

---

## 维度1：名称简化安全性

| 检查 | 逻辑 |
|------|------|
| 你的标签用名 vs 供应商原始名称 | 提取供应商配料表 |
| 供应商配料表只有1种成分 | ✅ 简化100%安全 |
| 供应商配料表有≥2种成分 | → 进入维度2 |

## 维度2：复合配料展开红线

| 条件 | 判定 | 依据 |
|------|:--:|------|
| 复合配料 + 成品占比>25% | 🔴 必须展开 | GB 7718-2025 4.3.3 |
| 复合配料 + 有国标/行标/地标 + 占比≤25% | ✅ 可不展开 | GB 7718-2025 4.3.3 |
| 复合配料 + 执行企标(Q/xxx) | ⚠️ 严格地区不认可企标豁免 | 企业标准≠国标/行标/地标 |
| 复合配料 + 无国标/行标/地标/地标 + 任意占比 | 🔴 必须展开 | |

## 维度3：添加剂带入判定

| 条件 | 判定 |
|------|:--:|
| 供应商配料含添加剂 + 该添加剂在成品中起工艺作用 | 🔴 需在成品标签标示 |
| 供应商配料含添加剂 + 在成品中不起工艺作用 + 符合带入原则 | ✅ 可免标 |

## 维度4：配料名称标准化

- 添加剂名称对照 GB 2760 标准名称
- 同一配料不同供应商 → 名称须统一

## 维度5：致敏原穿透

- 从供应商配料表提取致敏原清单
- 与成品标签致敏原标示交叉比对
- 漏标 → 🔴

## 维度6：客户禁用穿透

- 供应商所有配料 × 客户禁限用清单
- 精确匹配命中 → 🔴

## 维度7：供应商标签本身合规

- SC号有效、配料表规范、添加剂标准名、致敏原标示、GMO声明

## 维度8：添加量与排列顺序一致

- 供应商宣称含量 vs 成品配料表排列顺序
